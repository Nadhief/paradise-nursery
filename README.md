# Paradise Nursery

Paradise Nursery is a simple React-based e-commerce application for browsing and purchasing houseplants. Users can explore plants by category, add items to the shopping cart, update item quantities, remove items from the cart, and view the total number of plants and overall cost before checkout.

## Features

- Browse houseplants by category
- View plant images, names, and prices
- Add plants to the shopping cart
- Increase and decrease item quantities
- Remove items from the cart
- View total items and total cost
- Continue shopping
- Checkout placeholder ("Coming Soon")

## Built With

- React
- React Router
- Redux Toolkit
- CSS
- Vite
- Tabler Icons

## Getting Started

Install dependencies:

```bash
npm install
```

Run the development server:

```bash
npm run dev
```

Build the project:

```bash
npm run build
```
